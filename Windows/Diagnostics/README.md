# Windows — Diagnostics

General-purpose diagnostic scripts for Windows endpoints. Safe to paste into any remote terminal — RMM, RDP, or PowerShell remoting.

## Scripts

| Script | When to use |
|--------|-------------|
| `Get-EndpointHealth` | PC running slow |
| `Get-EventLogExport` | need to send logs to a vendor |
| `Get-RecentChanges` | something changed and now it's broken |
| `Get-SystemDetails` | what OS version is this PC running |
| `Get-GraphicsDriverHealth` | screen flickering |
| `Get-Win11Readiness` | can this PC upgrade to Windows 11 |
| `Get-PrintHealth` | printer not printing |
| `Get-DiskHealth` | low disk space warning |
| `Get-BrowserCacheHealth` | browser running slow |
| `Get-BatteryHealth` | laptop battery draining too fast |
